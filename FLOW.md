## 召回流程 (recall)

```
用户查询
    │
    ▼
┌─────────────────────────────────────────────────┐
│ 1. 边界检查                                      │
│    - db 是否已初始化                            │
└─────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────┐
│ 2. 自适应检索判断                                │
│    - shouldRetrieve()                          │
│    - 检查 forceKeywords                         │
│    - 检查 query.length >= minQueryLength       │
└─────────────────────────────────────────────────┘
    │                        ▲
    │ 不需要召回            │
    ▼                        │
┌─────────────────────────────────────────────────┐
│ 3. 缓存检查                                      │
│    - configHash + query 生成 key               │
│    - 命中则直接返回                             │
└─────────────────────────────────────────────────┘
    │                        ▲
    │ 命中                  │
    ▼                        │
┌─────────────────────────────────────────────────┐
│ 4. 数据库查询                                    │
│    - SELECT * FROM memories                   │
│    - 按 tier, importance, access_count 排序   │
│    - 预取 maxResults * 3 条                   │
└─────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────┐
│ 5. 时间衰减计算                                  │
│    - recencyDecay 指数衰减                     │
│    - weibullDecay Weibull 衰减（可选）         │
│    - reinforcement 强化因子                   │
└─────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────┐
│ 6. MMR 去重（可选）                             │
│    - mmrDeduplicate()                          │
│    - 保持多样性                                │
└─────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────┐
│ 7. 硬阈值过滤（可选）                           │
│    - hardMinScore 过滤                         │
└─────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────┐
│ 8. 截断结果                                      │
│    - 取前 maxResults 条                        │
└─────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────┐
│ 9. 写入缓存                                      │
│    - set(cacheKey, result)                    │
└─────────────────────────────────────────────────┘
    │
    ▼
✅ 返回 { hasMemory, memories }
```

## 定时任务

```
每天 00:00:00
    │
    ▼
┌─────────────────────────────────────────────────┐
│ cleanup() 执行                                  │
│                                                  │
│ 计算 cutoff = now - cleanupDays (180天)         │
└─────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────┐
│ DELETE FROM memories                            │
│ WHERE last_accessed < cutoff                    │
│   AND layer = 'general'                         │
│   AND tier = 'peripheral'                       │
└─────────────────────────────────────────────────┘
    │
    ▼
✅ 清理完成，日志记录删除数量
```
