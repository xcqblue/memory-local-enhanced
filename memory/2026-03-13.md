## 核心流程

### 存储流程
```
content → 规则分类 → LLM增强(可选) → XSS转义 → 哈希去重 → 智能去重(Jaccard) → 核心识别 → 写入
```

### 召回流程
```
query → 缓存 → 搜索 → 时间衰减 → 排序 → Token限制 → 返回
```

### 智能去重策略
- ≥98% → DUPLICATE (更新访问时间)
- ≥85% → UPDATE (合并内容)
- <85% → NEW (新写入)

## 备注

- 智能去重基于 Jaccard 相似度算法
- 时间衰减公式: score × (0.3 + 0.7 × 0.5^(t/90天))
- 核心记忆 (layer=core) 永久保留
- 定时清理保护核心记忆
